# Simulator basic idea of Overfitting and Underfitting along with Regularization methods

1. Overfitting

High polynomial degree

Low regularization

Training error ↓

Test error ↑

2. Underfitting

Low polynomial degree

Model too simple

Both errors high

3. L1 (Lasso)

Can shrink coefficients to zero

Performs feature selection

4. L2 (Ridge)

Shrinks coefficients smoothly

More stable model

Interactive Machine Learning simulator.

It demonstrates Train–Test data splitting, multiple classification models (Logistic Regression, SVM, Decision Tree, Random Forest), 

and evaluation metrics including Accuracy, Precision, Recall, F1-Score, Confusion Matrix, and AUC.

The simulator also includes L1 (Lasso) and L2 (Ridge) regularization to visualize overfitting and underfitting behavior. 

It is designed for educational purposes to help understand core machine learning concepts in a simple and interactive way.
